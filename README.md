# Markov Weather Simulator

This project is a web-based Markov Weather Simulator that models how weather changes over time using transition probabilities between states.

## Features
- Simulates weather over multiple days
- Uses probabilistic state transitions (Markov model)
- Allows users to edit transition probabilities
- Displays forecast with real dates

## AI Concepts
- Markov processes
- Probabilistic reasoning
- State transitions over time

## How to Run
Open index.html in a browser or run:
python3 -m http.server 8000
